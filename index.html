Τι είναι το CSS;

Το CSS (Cascading Style Sheets) είναι μια γλώσσα που χρησιμοποιείται για να περιγράψει την εμφάνιση ενός εγγράφου HTML.

Επιλογείς (Selectors)


/* Επιλογή όλων των παραγράφων */
p {
  color: blue;
}

Flexbox


.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Grid Layout


.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
